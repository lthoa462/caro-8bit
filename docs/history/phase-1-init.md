# Phase 1: Khởi tạo dự án

## Các công việc đã thực hiện:
- Thiết lập cấu trúc thư mục rõ ràng (`server`, `client`, `docs`).
- Tạo các thư mục con cho logic (controllers, models, sockets) và UI (components, hooks, assets).
- Chuẩn bị môi trường cho các giai đoạn tiếp theo.

## Ghi chú:
- Dự án sẽ sử dụng cấu trúc Monorepo đơn giản với `server` và `client` là hai package riêng biệt.
- Tài liệu kế hoạch và lịch sử được lưu trong `/docs`.
